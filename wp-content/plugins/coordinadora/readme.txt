== Description ==
Plugin oficial para integrar WooCommerce con Coordinadora Mercantil, podrás crear tus guías y hacer seguimiento de las mismas de una manera más ágil.

== Installation ==
1. Instale el plugin a través del Administrador de Plugins de su sitio o descárguelo y suba el archivo .zip.
2. Ingrese al Administrador de Plugins de su sitio y active el plugin.
3. Diríjase al menú principal izquierdo y encontrará un nuevo elemento llamado Coordinadora.
4. Al hacer clic será redirigido a nuestra plataforma.
5. Registrarse e ingresar a la plataforma para comenzar a realizar guías sobre tus pedidos.
