== Installation ==

<strong>From Your WordPress Dashboard</strong>
 
1. Go to Plugins ><strong>Add New</strong>
2. Search for <strong>Post Layouts for Gutenberg</strong>
3. Click on <strong>Install Now</strong> Button
4. Click on <strong>Activate Now</strong> After Installed in the Backend 
 
<strong>From WordPress.org</strong>
 
1. Download <strong>[Post Layouts for Gutenberg](https://wordpress.org/plugins/post-layouts/)</strong>
2. Upload the <strong>'post-layouts’</strong> folder to the /wp-content/plugins/ directory
3. Activate <strong>Post Layouts for Gutenberg</strong> plugin from your plugins page
