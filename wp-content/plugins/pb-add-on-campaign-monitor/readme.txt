== Description ==

Profile Builder - Campaign Monitor Integration Add-On

Easily associate Campaign Monitor client list fields with Profile Builder fields. Also, make use of the Profile Builder Campaign Monitor Widget to add more subscribers to your lists.


NOTE:

This plugin requires and is compatible with Profile Builder version 2.1.0 or higher

== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to Campaign Monitor tab under Profile Builder menu and enter a valid Campaign Monitor API key and then select a client from the drop-down.
4. Go to Manage Fields tab under Profile Builder menu and you'll have access to Campaign Monitor Subscribe field.
